# Dungeon Escape 

A 2D platformer game built in C++ using the Allegro 5 library.

## How to Play

1. Go to the `build/Release` folder
2. Double click `assignment.exe` to launch the game
3. Collect all coins and reach the door to escape!

## Controls

- Arrow keys to move
- Up arrow to jump
- F11 to minimze and maximize the screen

## Built With

- C++
- Allegro 5 library
- Visual Studio 2022

## Project Structure

- `src/` - All source code files
- `assets/` - Game images and sprites
- `build/Release/` - Playable game executable
